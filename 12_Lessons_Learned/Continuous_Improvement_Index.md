# Continuous Improvement Index

## Purpose

Studio-wide learning index for Village Stories.

Episode files record what one episode taught.  
**Category registers** collect repeatable patterns across episodes and tools.

| Layer | Home | Authority |
| --- | --- | --- |
| Episode evidence | `Episode_XXX.md` | Primary record of what happened |
| Category registers | `Categories/` | Cross-episode pattern library |
| Policy | Canon / Production / Standards / AI manuals | Only after promotion |

---

## Category registers

| Register | File | Collects |
| --- | --- | --- |
| Prompt failures | `Categories/Prompt_Failures.md` | Lock misses, gaze, style drift, negatives |
| Google Flow limitations | `Categories/Google_Flow_Limitations.md` | Tool boundaries and safe workarounds |
| Animation mistakes | `Categories/Animation_Mistakes.md` | Motion stacking, identity morph, camera errors |
| Visual consistency | `Categories/Visual_Consistency.md` | Character/location/object/style drift patterns |
| Thumbnail improvements | `Categories/Thumbnail_Improvements.md` | Mobile read, honesty, bait failures |
| SEO observations | `Categories/SEO_Observations.md` | Title/description/tag truths that work or fail |
| Production bottlenecks | `Categories/Production_Bottlenecks.md` | Throughput friction with evidence |
| Workflow improvements | `Categories/Workflow_Improvements.md` | Handoff / checklist / template wins |
| Viewer feedback | `Categories/Viewer_Feedback.md` | Audience signals after release |
| Quality improvements | `Categories/Quality_Improvements.md` | Gate/score patterns worth changing practice |
| Future recommendations | `Categories/Future_Recommendations.md` | Parked proposals awaiting standards/canon review |

---

## Intake rule

1. Write the episode entry first (`LL-EP###-##`) when evidence exists.
2. If the lesson is reusable, **also** add a short cross-link row in the matching category register.
3. Promote to standards/canon only via existing change processes.
4. No invented lessons. Empty registers are healthy until production produces evidence.

---

## Promotion path

```text
Episode entry
  → Category register (if reusable)
    → Standards / Prompt manual / Canon change (if policy needed)
```
