# Farming Tools

## Purpose

This folder permanently defines the practical farming tools used by Father, Neighbour Father, Village Farmers, and the wider agricultural community.

---

# Canonical Tool Set

Maintain:

* Wooden-handled sickle.
* Hoe / mamatti for soil work.
* Spade.
* Hand weeding tool.
* Bamboo or wooden carrying pole.
* Coir rope.
* Seed basket.
* Woven harvest basket.
* Irrigation channel-clearing tool.

Tools are hand-made, repaired, and maintained through the Blacksmith Forge and Carpenter Workshop where appropriate.

---

# Materials and Appearance

* Forged iron blades with natural dark patina.
* Local hardwood handles.
* Coir or natural-fibre rope.
* Bamboo and woven baskets.
* No bright plastic handles, branded machinery, or modern industrial equipment.

---

# Handling and Storage

Sharp or heavy tools are stored safely in the Livestock Shelter, outdoor work zones, or field contexts—not left in children's play areas.

Father uses tools with calm practiced body mechanics. Children observe or handle only small safe objects under direct supervision.

---

# Sounds

* Sickle through grass.
* Hoe into soil.
* Basket rustle.
* Coir rope pull.
* Tool handle set on packed earth.

---

# Continuity Rules

Never introduce tractors, combines, electric pumps, or modern agricultural machinery unless a future documented canon change explicitly adds them.

---

# Canon Authority

This document permanently defines Village Stories farming tools.

---

# Guiding Principle

Farm tools should look used by real hands: simple, durable, repaired, and inseparable from seasonal work.
