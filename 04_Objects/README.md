# Objects

## Purpose

This folder defines recurring physical objects in Village Stories.

Objects are permanent production assets. Each must retain stable materials, scale, use, storage, sound, and visual identity across Google Flow scenes.

# Object Continuity Rules

Always maintain:

* Early-1990s rural Tamil Nadu material culture.
* Hand-made and naturally used surfaces.
* Logical storage and handling.
* Stable recurring identity.

Never introduce modern branded, plastic-dominant, or anachronistic objects without a documented permanent change.

This folder permanently defines production objects within the Village Stories universe.
