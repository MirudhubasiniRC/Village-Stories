# Object Philosophy

These are hand-made, used, and maintained tools—not decorative props.

They show:

* Real material texture.
* Gentle signs of daily use.
* Stable size and shape.
* Traditional early-1990s rural Tamil Nadu craft.
* No modern plastic replacement unless separately introduced by permanent canon.
