# Location Reference Specs

## Purpose

Working briefs for generating **permanent location identity stills**.

Each brief summarizes locks already owned by `02_World/`.  
Do not treat this file as higher authority than the location bible.

**Style lock (all refs):** Studio Ghibli-inspired digital anime — painterly watercolor/gouache, soft matte, atmospheric depth. Early-1990s rural Tamil Nadu setting. Full-frame same rendering language (no photoreal backgrounds).

**Delivery intent:** Establishing or identity-wide shot, readable as “this place forever,” minimal or no featured characters unless needed for scale. Prefer empty or background-life-only frames for identity purity.

**Attachment role when used:** `LOCATION_IDENTITY` / `STYLE_TARGET`
