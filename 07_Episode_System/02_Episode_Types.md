# Episode Types

## Purpose

This manual classifies the repeatable episode constructions available to Village Stories.

An episode type is a production pattern, not a story doctrine or a list of interchangeable ideas.

It helps a writer, director, storyboard artist, Google Flow operator, and editor select the right physical shape for a canon-approved premise.

Use it after `06_Story_Engine/` has established that the premise is emotionally truthful and canon-safe.

Do not use a type to force a premise into a familiar formula.

Choose the type that lets the real action be seen most clearly.

## Guiding Principle

Select the smallest episode type that can show the full value of the moment.

## Authority

This document is the Episode System authority for type selection, format fit, and type-specific production logic.

It cannot approve an undocumented character, location, object, profession, custom, or modern detail.

Individual canon documentation remains authoritative.

The Story Engine remains authoritative for conflict, emotion, character truth, and story eligibility.

## How to Use Types

Every approved premise should be tested against four questions:

1. What is the audience watching change?
2. What makes that change visible on screen?
3. Which family member or members make the change emotionally legible?
4. Can the change resolve in a Short, or does it honestly require long-form observation?

The answer should identify a primary type.

It may identify one supporting type in long-form.

Do not combine several types merely to make an episode feel busy.

## Type Selection Rules

Choose a type based on the dominant on-screen action.

Choose a task-led type when physical progress is the clearest source of movement.

Choose a relationship-led type when a practical action reveals a change between family members.

Choose a season-led type when weather or annual timing changes the plan.

Choose a place-led type when an established location supplies the unique action, rhythm, or visual payoff.

Choose an observational type when a single recurring routine becomes meaningful through a precise condition.

Reject or revise a premise when its type cannot be named without inventing an event.

## Format Key

**Short** means a self-contained 60–120 second episode.

**Connected Shorts** means a naturally continuing event divided into self-contained parts.

**Long-form** means approximately five to ten minutes with a primary thread and, only when needed, one related supporting thread.

The format is a delivery choice.

It is not a measure of a premise’s importance.

## 1. Task Completion Episodes

### Construction

The episode follows a visible practical goal from an initial state to a changed state.

The goal may involve an established household, field, garden, repair, or errand activity that is documented elsewhere in the repository.

The episode opens on the incomplete task or the evidence that makes it necessary.

It then shows preparation, meaningful effort, a small adjustment, and a completed or responsibly paused state.

### Why It Works

Viewers understand progress without extensive explanation.

The materials, hands, tools, weather, and sounds create natural visual rhythm.

The final state provides a built-in payoff.

### Best Format

Use a Short when one stage has a distinct before-and-after result.

Use connected Shorts when documented process genuinely crosses days or stages.

Use long-form when preparation, work, and shared rest each contribute meaning.

### Production Rules

Show the initial state before removing it.

Show at least one decisive action, not only a montage of hands.

Keep object positions consistent between generated shots.

End on the result being used, received, observed, or set aside with purpose.

### Positive Signal

The viewer can describe what was accomplished without narration.

### Negative Signal

The task could be replaced by any generic “busy village” montage without changing the episode.

## 2. Cooking and Shared-Meal Episodes

### Construction

The visible process of preparing, cooking, serving, or sharing food carries the episode.

The emotional meaning comes through care, timing, resourcefulness, memory, or family participation.

The food and cooking methods must come from established world, object, culture, and location documentation.

### Why It Works

Cooking offers strong sensory progression: gathering, washing, grinding, heating, stirring, aroma, serving, and rest.

It can hold both a practical goal and a quiet relationship beat.

### Best Format

A Short should select one key stage and one payoff.

Long-form may show the complete arc only if each stage visibly changes the meal or the family’s experience.

### Production Rules

Do not turn preparation into a rapid recipe tutorial.

Do not introduce ingredients, tools, or techniques without canonical support.

Use heat, texture, steam, utensil sound, and serving arrangement as story evidence.

Avoid food glamour that erases the lived-in early-1990s rural Tamil Nadu setting.

### Edge Case

If the cooking is visually repetitive, place the emotional turn at a task transition rather than adding unrelated conversation.

## 3. Household Routine Episodes

### Construction

A familiar household routine becomes the episode because one condition makes it newly meaningful.

The condition may be weather, an object’s absence, a family member’s need, time pressure, or a remembered use.

### Why It Works

Routine gives the audience recognition.

The condition gives the routine an episode shape.

### Best Format

Usually a Short.

Use long-form only if several linked routine stages reveal the family’s shared rhythm without repetition.

### Production Rules

Open after the routine has begun, at the moment the meaningful condition becomes visible.

Use location sound and object handling to distinguish this from generic domestic footage.

End when the routine returns to balance or establishes a truthful new temporary rhythm.

### Negative Signal

If nothing differs from any other day, it is a texture study, not yet an episode.

## 4. Repair and Resourcefulness Episodes

### Construction

An established object, working surface, route, or household system requires patient attention.

The episode follows observation, diagnosis through action, careful work, testing, and an honest result.

### Why It Works

Repair naturally creates questions: what is wrong, what can be used, and will the practical fix hold?

It demonstrates care without turning ordinary difficulty into a crisis.

### Best Format

A Short can follow one small repair and test.

Connected Shorts can cover preparation, repair, and later use when each has its own resolution.

Long-form suits a documented repair process with multiple meaningful stages.

### Production Rules

Never invent technical expertise beyond character documentation.

Show the object’s condition clearly before repair.

Keep the object’s damage, parts, and repaired state continuous in Google Flow generations.

The final test must answer the question introduced at the opening.

### Edge Case

If the repair fails or must wait, let the episode resolve in adaptation, care, or a clear next practical step.

Do not manufacture disappointment merely to create a sequel.

## 5. Weather-Adaptation Episodes

### Construction

The weather alters an already plausible plan.

The family notices the condition, adjusts task order or location use, acts within the new condition, and finds a proportionate resting point.

### Why It Works

Weather gives the episode an immediate visible force while remaining truthful to village life.

It can change light, sound, surfaces, movement, and pace without requiring melodrama.

### Best Format

Shorts work when there is one clear adaptation.

Long-form works when the changing weather affects a sequence of connected work and rest.

### Production Rules

Verify weather, season, ground condition, clothing, daylight, and water behaviour against world documentation.

Show the evidence that makes the adaptation necessary.

Let weather sound remain audible where it is emotionally or physically meaningful.

Never imply severe danger unless the canon and Story Engine genuinely support it.

### Negative Signal

Rain is used only as a visual effect and changes nothing in the episode.

## 6. Seasonal Transition Episodes

### Construction

An established seasonal change changes available work, household preparation, travel, light, or shared expectation.

The episode observes the family’s response through a specific action.

### Why It Works

Seasonal continuity rewards returning viewers and makes the village feel like a living place rather than a static set.

### Best Format

Shorts can capture one seasonal sign and response.

Long-form can follow a preparation or process whose stages carry genuine seasonal meaning.

### Production Rules

Consult the season tracker before drafting.

Preserve vegetation, sky, mud, water, clothing, and object condition from the selected time.

Do not use a festival or seasonal label as decoration if no visible preparation or effect appears.

## 7. Family Cooperation Episodes

### Construction

Two or more established family members work toward one practical result.

The episode reveals their relationship through timing, help, observation, patience, or an adjustment in how they work together.

### Why It Works

Cooperation creates warmth and movement without requiring an argument.

It lets each character’s documented strengths affect the screen action.

### Best Format

Use a Short for one shared task and one recognisable exchange.

Use long-form when multiple task stages allow each participant to contribute without crowding.

### Production Rules

Do not assign roles by stereotype.

Check individual character documentation before selecting actions.

Make each contribution visible.

Avoid having one character explain another character’s feeling when an action can show it.

### Edge Case

If five family members are present, simplify the physical action so the viewer can follow who is doing what.

## 8. Gentle Sibling-Repair Episodes

### Construction

Elder Brother and Younger Daughter encounter a proportionate disagreement within a real activity.

The episode shows the practical cause, a pause or consequence, an honest adjustment, and a small repair.

### Why It Works

The activity keeps the relationship movement concrete.

The repair model remains humane and age-appropriate.

### Best Format

Use a Short for a single misunderstanding and repair.

Use connected Shorts only when each part resolves a separate stage: disagreement, reflection, and new cooperation.

### Production Rules

Do not turn either child into a villain.

Do not use humiliation, cruelty, or extended sulking as retention.

Keep adult intervention consistent with Mother’s and Father’s documented behaviour.

End on changed action, not a lecture.

### Negative Signal

The episode needs shouting, punishment, or a false accusation to feel dramatic.

## 9. Parent Memory and Continuity Episodes

### Construction

Mother or Father encounters an established object, recipe, place, routine, or sensory cue that connects present work to a remembered earlier moment.

The memory serves current family life rather than becoming romance-focused plot.

### Why It Works

It creates emotional depth through a physical bridge between then and now.

The present-day task keeps the episode grounded.

### Best Format

Shorts should use one trigger, one remembered feeling, and one present response.

Long-form may combine memory with a related current preparation or family act.

### Production Rules

Use visual association, sound, object handling, or a brief spoken recollection only when canon permits.

Clearly mark any time shift through lighting, costume, object state, or edit language.

Do not invent a detailed past timeline simply to deepen the episode.

## 10. Learning Through Observation Episodes

### Construction

Younger Daughter or Elder Brother observes, attempts, asks through action, practices, or understands a practical lesson.

The lesson occurs inside real family or village activity.

### Why It Works

Learning gives a task an internal progression while preserving the series’ non-lecturing tone.

### Best Format

Usually a Short.

Use long-form only when a real process contains several meaningful stages of learning.

### Production Rules

Show the lesson in action before naming it.

Let a small mistake be safe, proportionate, and repairable.

Do not make the adult infallible or the child incompetent for convenience.

Do not pause the story for an educational speech.

### Edge Case

If the lesson cannot be shown through a documented activity, it does not belong in the episode.

## 11. Family Dog Routine Episodes

### Construction

The Family Dog’s established daily relationship to a household movement, route, sound, or family member becomes a supporting visual thread.

The dog’s behaviour must remain plausible and non-verbal.

### Why It Works

The dog can add anticipation, gentle humour, companionship, and a recurring sensory cue.

### Best Format

Best as a Short or a secondary long-form thread.

### Production Rules

Never make the Family Dog rescue, solve, or understand a problem beyond realistic behaviour.

Maintain coat, size, markings, movement, and location continuity according to character documentation.

Use the dog to observe or accompany an existing action, not to replace it.

### Negative Signal

The episode works only because the dog behaves like a human character.

## 12. Location-Routine Episodes

### Construction

An established location’s documented rhythm creates the episode’s action.

The location may be the family home, a work area, a route, Tea Stall, Small Grocery Store, Primary School, Village Utilities, or another documented place.

The specific location documentation determines what can happen there.

### Why It Works

Familiar places become emotionally rich when their ordinary use is precise.

The audience learns to recognise their geography, sounds, objects, and daily tempo.

### Best Format

Shorts suit one visit, errand, or location-specific task.

Long-form suits movement between documented locations only when the route and transitions materially affect the episode.

### Production Rules

Read the location file before writing a scene.

Preserve fixed layout, approach route, signage, furnishings, soundscape, and permitted activities.

Use establishing evidence when cutting to a location after another setting.

Do not invent shop owners, customers, or social events unless already documented.

## 13. Errand and Preparation Episodes

### Construction

A necessary item, message, or preparation links home to one or more established locations.

The episode follows a clear purpose, route logic, acquisition or completion, and return or handoff.

### Why It Works

Errands create natural forward movement and allow place to shape the story.

### Best Format

Use a Short for one destination and one completed need.

Use long-form only where each movement stage changes the task or relationship.

### Production Rules

Do not use travel shots as filler.

Every route segment must establish distance, season, anticipation, or practical cause.

Verify who may travel, how they travel, and why against character and world documentation.

## 14. Festival and Special-Preparation Episodes

### Construction

The episode follows a documented preparation activity rather than treating a cultural event as visual spectacle.

The practical work reveals family rhythm, seasonal timing, care, or anticipation.

### Why It Works

Preparation offers clear process, cultural specificity, and a natural communal feeling while retaining the series’ scale.

### Best Format

A Short should show one preparation step with a clear result.

Long-form can join related preparations when the timeline and location logic are verified.

### Production Rules

Check Culture, season, location, object, and canon documents before use.

Do not invent rituals, costumes, crowds, foods, or public activity.

Avoid generic “festival montage” editing.

Show what the present family is doing and why.

## 15. Observation and Sensory Episodes

### Construction

A particular condition—light, rain, heat, wind, dawn work, an object sound, or a familiar pause—frames a small task or response.

The sensory condition cannot be the only content.

It must change how an established character acts, waits, works, or notices.

### Why It Works

This type supports the ASMR and visual identity of Village Stories while keeping the episode purposeful.

### Best Format

Most effective as a Short.

May form a breathing section inside long-form.

### Production Rules

Select one dominant sensory thread.

Keep music restrained so natural sound can carry the moment.

Do not stretch a sensory loop beyond its changing value.

End with a concrete action or settled image.

## Combining Types in Long-Form

Long-form may combine one primary type with one supporting type.

Examples of valid construction:

* A task-completion episode with family cooperation.
* A weather-adaptation episode with cooking or household routine.
* A seasonal-transition episode with a location-routine thread.
* A repair episode with a gentle learning thread.

The supporting type must change the meaning of the main thread.

It may not become a detached subplot.

Do not combine more than two types without explicit production approval.

## Connected Shorts Decision Rules

Use connected Shorts only when all conditions are true:

* The documented process cannot honestly fit one Short.
* Each part has a distinct action and a complete mini-payoff.
* A new viewer can understand the current part quickly.
* The next part follows a real temporal or practical continuation.
* The series does not withhold a simple answer to create a click.

Useful connected structures include:

* Preparation → completion → use.
* Notice → repair → test.
* Weather change → adaptation → return to work.
* Misunderstanding → reflection → changed cooperation.

Avoid “Part 1” when the first release contains no complete experience.

## Google Flow Guidance by Type

Before generating, identify the visual proof required by the type.

Task episodes need before, action, and after states.

Relationship episodes need spatial interaction, pauses, and changed behaviour.

Weather episodes need verifiable environmental continuity.

Location episodes need stable geography and documented fixed elements.

Sensory episodes need close, controllable actions and clean sound-planning opportunities.

Generate only shots that demonstrate these proofs.

Do not generate ornamental material first and attempt to find a story afterward.

## Type-Level Quality Gate

Approve a type choice only when:

* The primary action is canonical and visible.
* The format fits the action’s natural duration.
* The selected characters have a documented reason to participate.
* The location is established and used accurately.
* The season affects the episode where relevant.
* The required visual states can be preserved across Google Flow clips.
* The ending provides completion rather than a forced cliffhanger.

Revise the type when any one of these fails.
