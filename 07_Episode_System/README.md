# Episode System

## Purpose

This folder turns the rules in `06_Story_Engine/` into consistently structured, engaging episodes.

Story Engine defines how stories work. Episode System defines how each Short or long-form episode starts, holds attention, ends, and connects to what comes next.

---

# Core Rule

Every episode must be satisfying on its own while leaving a truthful reason to return: a continuing task, an unanswered gentle question, a relationship feeling, or tomorrow's practical need.

Never use false cliffhangers, danger, or withheld resolution simply to force retention.
