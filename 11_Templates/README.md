# 11_Templates — Production Working Forms

## Purpose

Reusable fill-in forms for live episode production.

These are **working templates**, not doctrine manuals.

Copy a template into the relevant `10_Episodes/ep_XXX/` folder, rename per Naming Standards, fill the blanks, then approve.

## Authority

Templates do not invent facts.

## How to use

1. Copy template → episode folder.
2. Rename: `episode-ep_XXX__[artifact]__[state]__r01.md`
3. Replace every `[bracket]` field.
4. Link canon sources; do not paste canon.
5. Mark status: `draft` → `in_review` → `approved`.

## Owner

Templates Owner (Creative Director / Production Architect).

Do not expand this folder with manuals. Improve a template only when live production proves a blank is missing or unclear.
