# Community Character Groups

## Purpose

This folder defines recurring village groups that enrich Village Stories without requiring every background person to receive a personal name.

These are functional, visually consistent community presences. A member may later be promoted into an individual character only when repeated story importance earns that depth.

# Group Continuity Rules

Always maintain:

* Early-1990s rural Tamil Nadu authenticity.
* Distinct age, work, and social roles.
* Natural recurring background behaviour.
* No modern clothing, devices, or random visual redesign.

Never turn groups into anonymous repetitive crowds. Every group should feel varied but recognizably local.

---

# Canon Authority

These documents permanently define community group behaviour and presentation within the Village Stories universe.

---

# Guiding Principle

The community should feel like a village full of familiar lives, not a set of interchangeable background extras.
