# Potter

## Purpose

This folder contains the canonical documentation for the village Potter, a recurring artisan who works at the permanent Potter Clay Yard.

---

# Role

The Potter makes practical clay vessels, lamps, and festival diyas through hand-wheel, sun-drying, and outdoor-kiln work.
