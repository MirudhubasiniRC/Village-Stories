# Blacksmith

## Purpose

This folder contains the canonical documentation for the village Blacksmith, a recurring artisan who works at the permanent open-sided Blacksmith Forge.

# Character Role

The Blacksmith makes, sharpens, and repairs essential metal tools for farming and everyday village life.
