# Barber

## Purpose

This folder contains the canonical documentation for the village Barber.

He works beneath the permanent large shade tree on the west-side veranda-style barber stand.

---

# Character Role

The Barber provides haircuts, shaving, and simple grooming with skill, cleanliness, gentle conversation, and period-correct tools.
