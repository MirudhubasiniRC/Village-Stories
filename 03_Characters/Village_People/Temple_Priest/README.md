# Temple Priest

## Purpose

This folder contains the canonical documentation for the Temple Priest, a recurring village character who cares for the humble Vinayagar Shrine beneath the Sacred Banyan Tree.

---

# Character Role

The Temple Priest performs daily pooja, tends lamps and flowers, receives offerings, and helps the shrine remain peaceful and cared for.

He is a spiritual guide through quiet service, never a mystical spectacle or dominant authority.
